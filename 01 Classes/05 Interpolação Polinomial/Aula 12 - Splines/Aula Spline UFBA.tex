Exercícios:

Pontos tabelados para a função f(x)
 x    -2 -1 0 1 2
f(x)   0  4 2 0 4

Resposta: Sistema Spline Linear
S1(x) =  4x + 8, -2 <= x <= -1 
        -2x + 2, -1 <= x <= 0
        -2x + 2,  0 <= x <= 1
         4x - 4,  1 <= x <= 2
